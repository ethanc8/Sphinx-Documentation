1.1.6 How can I get GNUstep?

Many distributions include packaged versions of GNUstep (Debian, etc). To compile from sratch, download the GNUstep Startup package or get the HOWTO from gnustep-howto_toc.html. Get the latest releases from ftp://ftp.gnustep.org/pub/gnustep/core.
