DKPortNameRegistrationStatus

typedef NS_ENUM(uint32_t) DKPortNameRegistrationStatus;
The status returned by D-Bus in response to a request to register a name.
DKPortNamePrimaryOwner
The port is now the primary owner of the name.
DKPortNameQueued
The name has been queued and will be assigned to the port once previous registrands of the name go away.
DKPortNameExists
The name is already in use by a different port, but queuing was not requested.
DKPortNameAlreadyOwner
The port had already been registered by the port. No changes occurred.
