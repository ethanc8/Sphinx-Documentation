CWURLName

Discussion

This class provides support for handling POP3, IMAP and local URL schemes. The POP3 URL scheme is described in RFC 2384. The IMAP URL scheme is described in RFC 2192 while the local URL scheme is a simple Pantomime scheme to refer to local mailboxes.
