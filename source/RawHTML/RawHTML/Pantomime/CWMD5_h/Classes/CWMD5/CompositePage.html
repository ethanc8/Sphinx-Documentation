CWMD5

MD5 Message-Digest algorithm implementation.

Discussion

This class provides an easy interface to the MD5 Message-Digest algorithm. This algorithm is described in RFC 1321. It also supports the HMAC keyed-hashing for message authentication (described in RFC 2104). HMAC is used in the CRAM-MD5 SASL authentication mechanim.
