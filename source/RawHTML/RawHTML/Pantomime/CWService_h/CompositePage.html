split_lines

static inline NSData *split_lines(
    NSMutableData *theMutableData)
Parameters
theMutableData

The data from which to split lines from.

Return Value

A line as a NSData instance, nil if no line was splitted.

Discussion

This function is used to split lines from theMutableData and return the first one found immediately (by removing it first from theMutableData). IMAP, POP3 and SMTP servers reponses are always ended by CRLF.
