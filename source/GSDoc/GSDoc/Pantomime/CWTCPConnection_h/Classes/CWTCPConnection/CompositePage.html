CWTCPConnection

Discussion

This class, which implements the CWConnection protocol, offers an elegant interface to a TCP connection. See the documentation of the CWConnection protocol in order to see all the methods one can use on an instance of this class. Normally, you should never have to use this class or one of its methods directly.

isSSL

- (BOOL) isSSL;
Return Value

YES if using SSL, NO otherwise.

Discussion

This method is used to verify if the connection TCP connection is using SSL or not.

startSSL

- (int) startSSL;
Return Value

0 on success, < 0 on error.

Discussion

This method is used to start the SSL handshaking on an altready established TCP connection.
