addressesForName:background:

- ( NSArray *) addressesForName:
        (NSString *) theName  background:
        (BOOL ) theBOOL;

This method is used to obtain an array of IP addresses from a fully qualified domain name.

Parameter Descriptions
theName
The fully qualified domain name.
theBOOL
If YES, the call is non-blocking and returns nil. Otherwise, the call is block and returns an array of addresses, if any.
method result
The array of addresses encoded as NSData instances.
