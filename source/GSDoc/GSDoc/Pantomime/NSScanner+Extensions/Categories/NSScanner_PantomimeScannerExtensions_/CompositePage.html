NSScanner (PantomimeScannerExtensions)


Abstract

Pantomime extensions to NSScanner.

Discussion

This category provides useful extensions to the default NSScanner class.

Methods

scanUnsignedInt:

scanUnsignedInt:

- ( BOOL ) scanUnsignedInt:
        (unsigned int *) theValue;

This method is used to scan an unsigned int.

Parameter Descriptions
theValue
The value in which the unsigned int will be stored.
method result
YES if we find a decimal unsigned int, no otherwise.
