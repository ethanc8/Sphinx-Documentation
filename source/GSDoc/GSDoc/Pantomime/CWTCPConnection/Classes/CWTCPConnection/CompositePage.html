CWTCPConnection


Discussion

This class, which implements the CWConnection protocol, offers an elegant interface to a TCP connection. See the documentation of the CWConnection protocol in order to see all the methods one can use on an instance of this class. Normally, you should never have to use this class or one of its methods directly.
