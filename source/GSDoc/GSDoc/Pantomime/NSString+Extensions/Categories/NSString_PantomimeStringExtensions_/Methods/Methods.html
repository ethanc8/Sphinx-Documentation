modifiedUTF7String

- ( NSString *) modifiedUTF7String;

This method returns a newly created NSString object which can be converted using the NSASCIIStringEncoding using the modified UTF-7 encoding described in RFC3501. A modified UTF-7 string MUST be used in IMAP for maibox names.

method result
A modified UTF-7 string.
