UID

- ( NSUInteger ) UID;

This method is used to obtain the UID of a message. See 2.3.1.1. Unique Identifier (UID) Message Attribute of RFC 3501 for a detailed description of the UID attribute of a message.

method result
The UID, 0 if the receiver has no UID.

rawSource

- ( NSData *) rawSource;

IMAP specific implementation of the rawSource method. This method is always non-blocking. It might return nil if the raw source of the message hasn't yet been fetched from the IMAP server. The IMAPStore notifies the delegate when the fetch has been completed.

method result
The raw source of the message, nil if not yet fully fetched.

setFlags:

- ( void ) setFlags:
        (CWFlags *) theFlags;

This method overrides the one found in CWMessage. It will replace all the flags on the server for this message with the flags specified in the argument.

Parameter Descriptions
theFlags
The new flags for the receiver.

setUID:

- ( void ) setUID:
        (NSUInteger ) theUID;

This method is used to set the UID of the receiver. Normally, you shouldn't invoke this method directly.

Parameter Descriptions
theUID
The UID of the message.
