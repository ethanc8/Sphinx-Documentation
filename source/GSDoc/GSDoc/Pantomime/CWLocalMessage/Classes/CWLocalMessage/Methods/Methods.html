filePosition

- ( long int ) filePosition;

This method is used to obtain the position in the mbox file from which the message begins at. This method isn't really helpful if -type returns something other than PantomimeFormatMbox.

method result
The position in the file.

mailFilename

- ( NSString *) mailFilename;

This method is used to obtain the associated filename of the receiver, if -type is PantomimeFormatMaildir.

method result
The filename, nil if none was defined or if -type returns something other than PantomimeFormatMaildir.

setFilePosition:

- ( void ) setFilePosition:
        (long int ) theFilePosition;

This method is used to set the position in the mbox file from which the message begins at. This method isn't really helpful if -type returns something other than PantomimeFormatMbox.

Parameter Descriptions
theFilePosition
The position in the file.

setMailFilename:

- ( void ) setMailFilename:
        (NSString *) theFilename;

This method is used to set the associated filename for the receiver. This is useful and required if -type returns PantomimeFormatMaildir.

Parameter Descriptions
theFilename
The name of the file.
