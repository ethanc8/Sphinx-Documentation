NSMutableData (PantomimeExtensions)


Abstract

Pantomime extensions to NSMutableData.

Discussion

This category provides useful extensions for handling NSMutableData objects.

appendCFormat:

- ( void ) appendCFormat:
        (NSString *) theFormat, .
    .
    .
    ;

This method is used to append the variable arguments to the receiver. The bytes of the arguments must be composed only of ASCII characters.

Parameter Descriptions
theFormat
The format of the rest of the argument, as this method is a variadic method.

appendCString:

- ( void ) appendCString:
        (const char *) theCString;

This method is used to append the C string to the receiver.

Parameter Descriptions
theCString
The C string to append.

insertCString: atIndex:

- ( void ) insertCString:
        (const char *) theCString  atIndex:
        (NSUInteger ) theIndex;

This method is used to insert the C string into the received at the specified index. If theIndex is zero, the C string will be added at the beginning of the receiver. If theIndex is greater than the receiver's length, the C string will be appended to the receiver.

Parameter Descriptions
theCString
The C string to insert into the receiver.
theIndex
The index where to insert the C string.

replaceCRLFWithLF

- ( void ) replaceCRLFWithLF;

This method destructively elides all carriage return (CR, Control-M, ASCII 13) from the receiver.

replaceLFWithCRLF

- ( NSMutableData *) replaceLFWithCRLF;

This method is used to replace all occurences of line feed characters by sequences of a carriage return followed by line feed characters in the receiver.
