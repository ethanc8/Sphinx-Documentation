CWFlags


Abstract

This class provides an interface to message flags.

Discussion

This class provides methods to deal with message flags. Each message has a set of flags, like answered, deleted, seen, unread, etc. See the PantomimeFlag enum for the list of possible values.

add:

        (PantomimeFlag ) theFlag;

This method is used to add (combine) an additional flag to an existing set of flags.

Parameter Descriptions
theFlag
The flag to add. This value is one of the values of the PantomimeFlag enum.

addFlagsFromData:

- ( void ) addFlagsFromData:
        (NSData *) theData  format:
        (PantomimeFolderFormat ) theFormat;

This method is used to add flags contained in a NSData instance. Such flags are come normally from parsing a mailbox that contains Status or X-Status headers or from the maildir's info.

Parameter Descriptions
theData
The NSData instance from which to parse the flags.
theFormat
The format we are parsing.

This is the designed initializer for the CWFlags class.

maildirString

- ( NSString *) maildirString;

This method is used to return a string value of the flags. This representation is used by the maildir format. It corresponds to the info semantics and is equivalent to the Status field of the mbox format. The returned string always begins with "2,".

method result
A string value of the flags.

remove:

- ( void ) remove:
        (PantomimeFlag ) theFlag;

This method is used to remove a flag from the receicer's set of flags. The possibile values are part of the PantomimeFlag enum.

Parameter Descriptions
theFlag
The flag to remove from the set of flags.

removeAll

- ( void ) removeAll;

This method is used to remove all flags from the receiver.

replaceWithFlags:

- ( void ) replaceWithFlags:
        (CWFlags *) theFlags;

This method is used to replace all flags from the receiver with the ones specified in theFlags.

Parameter Descriptions
theFlags
The CWFlags instance which holds the new set of flags.
